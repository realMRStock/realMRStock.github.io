Slow-emission liquid scintillators for hybrid Cherenkov/scintillation

We designed a bi-solvent liquid scintillator with deliberately slowed scintillation kinetics to better separate Cherenkov and scintillation light in time. The study details optical transmission and light yield, emission spectra and time-profile measurements for different particles, wavelength-shifter optimization, and DIN purification via chromatography and fractional vacuum distillation. Discrimination performance is demonstrated using pulsed-source data. My contribution: led the data analysis and model fitting of the scintillation time profiles, including convolution-based timing models and uncertainty propagation. JINST 19 (2024) P09015

Water-based liquid scintillator (WbLS) with Triton-based surfactant

We formulated a stable WbLS using a Triton-based surfactant/antioxidant system, enabling an aqueous medium that supports both scintillation and Cherenkov detection. The paper reports optical transmission, light yield, and detailed scintillation time-profile measurements, and compares particle-identification performance with conventional organic LS. My contribution: led the time-profile data analysis and model fitting, harmonizing the timing model across WbLS and organic LS and validating systematics. JINST 19 (2024) P09008

JUNO: status and prospects

Overview of JUNO’s detector concept and physics reach: precise reactor-neutrino spectroscopy for mass ordering and oscillation parameters, plus sensitivity to supernova and solar neutrinos as part of a broad physics program. SciPost Phys. Proc. 17, 020 (2025)

Broad-line regions in active galactic nuclei

VLTI/GRAVITY spectro-interferometry of the quasar 3C 273 resolves sub-parsec rotation in the broad-line region. Photocenter shifts across the Pa-α line map a velocity gradient consistent with a gravitationally bound disk, yielding a black-hole mass of order 108–109 M⊙ and a BLR scale consistent with reverberation mapping. Nature 563, 657–660 (2018)

GitHub

I will publish compact Python utilities for day-to-day analysis:

  • Plotting style examples: styles sheet using matplotlib with examples for publication-ready figures (axes, fonts, color cycles, and error bars).
  • Fitting scripts: Wrappers around scipy and iminuit for common 1D models (Gaussian peaks, exponentials, uniform distribution) plus uncertainty reporting.

Each example will come with a reproducible notebook and a PNG output for quick reference.

Zenodo datasets

I plan to provide datasets and derived results relevant to liquid scintillator research:

  • Emission spectra of solvents, solutes, and wavelength shifters (CSV file format; wavelength in nm, intensity normalized).
  • Scintillation time profile model results for several mixtures (classical, slow organic, water-based), including fit parameters, uncertainties, and model definitions.

Each record will include a short README (provenance, measurement conditions, units) and a suggested citation. Where appropriate, GitHub releases will be cross-linked to Zenodo DOIs.

Under construction: first code and datasets will be posted incrementally.
